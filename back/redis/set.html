    # set

    set: 是一个无序的集合，集合中元素都是唯一的。set 对象的编码可以是 intset 或者 hashtable;inset 编码的集合对象使用的是整数集合作为底层实现。而底层是字典时，只不过所有的 value 都是 null。

    # 4.1 intset （整数集合）

    intset的内存是连续的，所有的数据增删改查操作都是在内存地址偏移的基础上进行的，并且整数的保存也是有序的，intset的内存示意图如下： 

    #define INTSET_ENC_INT16 (sizeof(int16_t))   //16位，2个字节，表示范围-32,768~32,767
#define INTSET_ENC_INT32 (sizeof(int32_t))   //32位，4个字节，表示范围-2,147,483,648~2,147,483,647
#define INTSET_ENC_INT64 (sizeof(int64_t))   //64位，8个字节，表示范
typedef struct intset {
    // 编码方式，一个元素所需要的内存大小，INTSET_ENC_INT16，INTSET_ENC_INT32，INTSET_ENC_INT64三种类型
    uint32_t encoding;
    // 集合长度
    uint32_t length;
    // 集合数组:按照从小到大有序排列的，并且元素不重复
    int8_t contents[];
} intset;

    # 4.3 intset 编码转换 hashtable 条件

    当 set 对象可以同时满足以下条件，则为 intset 编码：

    1. set 对象保存的所有元素都是整数
    2. set 对象保存的元素数量不超过 512个； 不满足上述情况，则使用 hashtable 编码。
